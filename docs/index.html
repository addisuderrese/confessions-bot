Step-by-Step Setup Guide

Prerequisites

Step 1 of 4

Gather Your Credentials

Before you begin, collect these key pieces of information. Click each item when you have it ready.

Telegram Bot Token

Get this from @BotFather on Telegram by creating a new bot.

Admin Telegram ID

Your personal user ID. Get it from @userinfobot.

Channel ID

The ID of your public channel where confessions will be posted. Usually starts with -100. The bot must be an admin.

PostgreSQL Database URL

A connection string to your database. Free tiers available from Aiven or Supabase.

Python 3.9+ Installed

Make sure you have a modern version of Python on your system.

Step 2 of 4

Get the Bot's Code

Download the code from GitHub and install the necessary Python packages.

git clone https://github.com/addisuderrese/confessions-bot.git
cd confessions-bot
pip install -r requirements.txt

Tip: Using a virtual environment is recommended. Run python -m venv venv first, then activate it before installing packages.

Step 3 of 4

Configure Your Bot

Create a file named .env in the main directory and add these values:

BOT_TOKEN="123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"
ADMIN_ID="987654321"
CHANNEL_ID="-1001234567890"
DATABASE_URL="postgres://user:password@host:port/dbname?sslmode=require"

Important: Never commit your .env file to Git. It contains sensitive credentials.

Step 4 of 4

Launch the Bot

With the configuration complete, you're ready to start the bot.

python main.py

On its first run, the bot will automatically create all necessary database tables. You should see log messages indicating a successful connection.

Your bot is now live

Try sending it a /start command on Telegram to verify it's working.

Commands & Usage

User Commands

/start

Shows welcome message. If used via a deep link (e.g., from channel), opens the specified confession with its comments.

/confess

Start a new anonymous confession. Select 1-3 categories, then write your text (10-3900 characters). Submission is queued for admin review.

/profile

View your ðŸ… Aura points, browse paginated lists of your confessions and comments, and request deletion of your posts.

/help

Displays the help message with all available commands and a quick guide to comment interactions.

/rules

View the full community rules and guidelines at any time.

/privacy

View privacy information about how your data is stored and used, with a link to the full Privacy Policy.

/cancel

Cancel the current action or form (confession submission, contact admin, etc.) and return to the main menu.

Admin Commands

/id <user_id>

Lookup user details: Telegram username, first name, Aura points, confession count, comment count, rule acceptance status, and block status.

/warn <user_id> <reason>

Send a formal warning message to a user. The reason is included in the warning notification.

/block <user_id> <duration> [reason]

Temporarily block a user. Duration format: 7d (7 days), 2w (2 weeks). Optional reason is shown to the user.

/pblock <user_id> [reason]

Permanently block a user from the bot. They will see the block message on every interaction attempt.

/unblock <user_id>

Remove a block (temporary or permanent) from a user, restoring their access to the bot.
